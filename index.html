What does HTML stand for, and what is its primary purpose in web development?

HTML stands for HyperText Markup Language. Its primary purpose is to structure the content of a webpage, defining the various elements such as headings, paragraphs, images, links, and more.

What is CSS used for in web development?

CSS stands for Cascading Style Sheets. It is used to control the presentation and layout of HTML elements on a webpage. With CSS, you can apply styles like colors, fonts, spacing, and positioning to make your web pages visually appealing.
